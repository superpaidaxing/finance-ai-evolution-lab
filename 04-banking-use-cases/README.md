# 04 银行 AI 用例库

这里用于长期积累银行 AI 场景。每个用例都应当能帮助自己回答：

- 这个场景真实存在吗？
- 痛点是否足够强？
- AI 是否比传统系统更适合？
- 数据是否可得？
- 风险是否可控？
- 是否有明确业务收益？

## 用例成熟度

- 想法：只是想法
- 已调研：已查资料
- 已设计：已有方案
- 已实验：做过实验
- 已验证：有结果或复盘
- 已发布：形成公开输出
